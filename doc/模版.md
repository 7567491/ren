# 科技产品广告 Prompt 最佳模版

基于2025年最新AI广告生成实践整理的5个高转化率模版

---

## 模版1: 短视频脚本生成型（推荐用于抖音/小红书）

```
角色：资深短视频营销专家，擅长科技产品推广

任务：为[产品名称]创作一个30秒短视频脚本

产品信息：
- 产品类型：[智能手表/耳机/手机/智能家居等]
- 核心卖点：[3个核心功能/优势]
- 目标人群：[年龄段、职业、兴趣]

脚本要求：
1. 开头3秒用痛点提问引发共鸣（如："经常找不到手机？"）
2. 中间展示产品核心功能和使用场景
3. 结尾强CTA（call-to-action），引导下单/关注
4. 总字数控制在150-200字（配音时长30秒）
5. 风格：[轻松幽默/科技感强/温馨治愈]

输出格式：
- 画面描述（镜头1-5）
- 旁白文案
- 背景音乐建议
- 字幕提示
```

**适用场景**: 抖音、快手、小红书等短视频平台
**优势**: 结构化输出，直接可用于视频制作

## 模版2: 产品视觉广告生成型

```
生成一张[产品类型]的产品广告图

产品描述：[产品名称]，[核心特性]

视觉要求：
- 构图：[产品]放置在[具体表面/背景]，采用[俯视/45度角/特写]视角
- 场景：[咖啡厅桌面/科技感工作台/极简白色背景/自然光窗边]
- 光照：[柔和自然光/戏剧性侧光/科技蓝光/暖色调灯光]
- 氛围：[专业商务/年轻活力/温馨居家/未来科技]
- 色彩方案：[主色调]+[辅助色]，[高对比度/低饱和度莫兰迪色系]

参考风格：[苹果极简主义/小米科技感/华为商务风/索尼精致感]

输出尺寸：[1080x1080方图/1920x1080横图/1080x1920竖图]
```

**适用场景**: 电商主图、社交媒体广告、产品展示
**优势**: 精确控制视觉效果，适合品牌调性统一

## 模版3: 高转化广告文案生成型

```
身份：[科技产品类目]营销文案专家

目标：为[产品名称]撰写高转化广告文案

产品核心信息：
- USP（独特卖点）：[与竞品的核心差异]
- 目标受众：[性别、年龄、职业、痛点]
- 使用场景：[通勤/健身/办公/学习]

文案结构（AIDA模型）：
1. Attention（吸引注意）：用反问/数据/痛点开头
2. Interest（激发兴趣）：展示产品3大核心功能
3. Desire（刺激欲望）：使用场景+用户获益
4. Action（促成行动）：限时优惠+立即下单

要求：
- 标题：15字内，包含核心关键词
- 正文：80-120字
- CTA：明确行动指令（如"限时立减200，点击抢购"）
- 风格：[专业理性/感性温暖/幽默风趣/紧迫感营销]

输出3个版本供A/B测试
```

**适用场景**: 电商详情页、信息流广告、朋友圈广告
**优势**: 遵循经典营销理论，转化率高

## 模版4: 多模态广告内容套装型（Meta AI风格）

```
为[科技产品]生成完整广告套装

输入信息：
- 产品图片：[上传产品照片]
- 产品名称：[具体型号]
- 核心特性：[技术参数/功能亮点]
- 预算：[$金额]
- 投放渠道：[Facebook/Instagram/微信/微博]

生成内容包括：
1. 5个不同版本的广告文案（标题+正文+CTA）
2. 3张视觉创意图（不同场景/风格）
3. 1个15秒短视频脚本
4. 受众定位建议（年龄/兴趣/地域）
5. 投放策略（时段/预算分配）

风格一致性：
- 品牌调性：[年轻时尚/专业可靠/创新前卫]
- 色彩方案：[品牌主色]为主
- 语言风格：[简洁直白/情感化/数据驱动]
```

**适用场景**: 多渠道整合营销活动
**优势**: 一键生成全套素材，保持风格统一

## 模版5: 痛点-解决方案故事型（推荐用于长视频）

```
角色：资深产品营销策划

任务：为[科技产品]创作一个60-90秒故事化广告脚本

故事结构（起承转合）：
1. 起（痛点场景）：展示用户日常遇到的具体困扰
   - 场景：[办公室/家里/通勤路上]
   - 痛点：[效率低/体验差/不方便]
   - 情绪：焦虑、烦躁、无奈

2. 承（问题加剧）：痛点带来的连锁影响
   - 后果：[错过deadline/影响心情/浪费时间]

3. 转（产品出场）：自然引入产品解决方案
   - 展示方式：[朋友推荐/偶然发现/主动搜索]
   - 核心功能：[3个关键特性演示]

4. 合（美好结局）：使用产品后的改变
   - 效果：[具体数据/场景对比]
   - 情绪：满意、自信、愉悦
   - CTA：引导购买/了解更多

输出要求：
- 分镜脚本（5-8个镜头）
- 旁白文案（情感化表达）
- 画面描述（具体到人物表情、环境细节）
- 背景音乐情绪（从低沉到明快的转变）
```

**适用场景**: B站、YouTube、品牌宣传片
**优势**: 情感共鸣强，品牌记忆度高

## 使用建议

### 选择指南
- **快速起量** → 模版1（短视频脚本）
- **视觉为主** → 模版2（产品图生成）
- **文案驱动** → 模版3（AIDA文案）
- **整合营销** → 模版4（多模态套装）
- **品牌建设** → 模版5（故事化长视频）

### 优化技巧
1. **具体化**：避免"高端""优质"等空洞词汇，用具体数据和场景
2. **本地化**：根据目标市场调整语言风格（如美国市场偏理性，中国市场偏感性）
3. **A/B测试**：同一模版生成3-5个版本，根据数据选出最优
4. **迭代优化**：根据投放效果反馈调整prompt细节

### 成本控制
- 文案生成：$0.001-0.005/次
- 图片生成：$0.02-0.15/张
- 视频生成：$0.15-0.50/条（根据时长和分辨率）
