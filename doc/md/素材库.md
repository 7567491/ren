# 广告素材图像/视频库资源汇总（2025）

## 一、免费素材库

### 国际平台

#### 1. **Pexels**
- **类型**：图片 + 视频
- **素材量**：超过300万张图片和视频
- **授权**：CC0许可，完全免费商用
- **特点**：高质量素材，无需注册即可下载，无需署名
- **网址**：pexels.com

#### 2. **Pixabay**
- **类型**：图片 + 视频 + 矢量图
- **素材量**：超过270万张图片和数千视频
- **授权**：CC0许可，免费商用
- **特点**：涵盖自然风景、人物肖像、城市建筑、科技艺术等主题
- **网址**：pixabay.com

#### 3. **Unsplash**
- **类型**：图片为主
- **素材量**：超过300万张高分辨率图片
- **授权**：免费，无需署名
- **特点**：摄影作品质量高，美学性强
- **付费版**：Unsplash+ $7/月（年付）或 $20/月（月付），提供独家内容
- **网址**：unsplash.com

#### 4. **Coverr**
- **类型**：视频
- **格式**：HD视频（MP4格式）
- **授权**：完全免费，无需注册，无需署名
- **特点**：专注视频素材
- **网址**：coverr.co

#### 5. **Mixkit**
- **类型**：视频 + AE模板
- **特点**：艺术性强的实拍视频，每周更新，提供"音乐+视频"组合功能
- **授权**：免费商用
- **网址**：mixkit.co

### 国内平台

#### 6. **光厂**
- **类型**：视频素材
- **素材量**：超过15万个4K/8K实拍片段
- **内容**：城市宣传、中国文化、科技特效等
- **授权**：标准化CC0授权
- **特点**：智能标签系统，国内领先的视频素材平台
- **网址**：guangchang.com

#### 7. **稻虎网**
- **类型**：短视频素材
- **特点**：4K高清无水印素材，面向自媒体和副业创作者
- **网址**：daohuhu.com

#### 8. **包图网**
- **类型**：综合素材（视频、音乐、图片、PPT模板等）
- **收费模式**：会员制，会员可免费下载
- **网址**：ibaotu.com

## 二、付费素材库

### 1. **Shutterstock**
- **素材量**：超过2亿件内容（图片+视频）
- **定价方案**：
  - 月付计划：$49/月（10张图片），年付优惠至 $24.91/月
  - 3张图片套餐：$43.99/月
  - 批量购买：低至 $0.22/张
- **特点**：业界巨头，素材全面，价格灵活
- **网址**：shutterstock.com

### 2. **Getty Images**
- **素材量**：超过7.5万名全球贡献者
- **定价方案**：
  - 单张图片：$199 - $499
  - 批量购买：$130 - $299/张
  - 特定用途授权（Rights-Managed）：$79起
  - 企业定制方案：按需定价
- **特点**：高端专业素材，独家内容，适合大型企业
- **备注**：2025年1月宣布与Shutterstock合并
- **网址**：gettyimages.com

### 3. **Adobe Stock**
- **定价方案**：
  - 入门计划：$29.99/月（年付，1个视频/月）
  - 高级计划：$199.99/月
- **特点**：与Adobe Creative Cloud深度整合
- **网址**：stock.adobe.com

### 4. **Storyblocks**
- **素材量**：超过150万视频、音频和图片
- **定价方案**：
  - 基础视频计划：$19/月 或 $99/年
- **特点**：订阅制，性价比高
- **网址**：storyblocks.com

## 三、免费 vs 付费对比

| 对比项 | 免费素材库 | 付费素材库 |
|--------|-----------|-----------|
| **价格** | 完全免费 | $0.22 - $499/张不等 |
| **素材独特性** | 使用广泛，容易撞素材 | 独家内容，使用率低 |
| **授权复杂度** | 简单（多为CC0） | 复杂（需注意使用范围） |
| **素材质量** | 中高质量 | 高端专业质量 |
| **适用场景** | 个人项目、小企业、自媒体 | 大型企业、品牌广告、商业项目 |

## 四、选择建议

**选择免费素材库，如果你：**
- 预算有限或刚起步
- 做自媒体、个人博客、小型项目
- 需要快速获取素材，不介意素材重复使用

**选择付费素材库，如果你：**
- 需要独特性和专业性
- 做品牌广告、大型商业项目
- 需要明确的商业授权保障
- 预算充足，追求高质量内容
